# Wiffle = ((640 - 3x) / 4)

### Breakpoint info

#### Mobile:

* **[A]LPHA**
    * `viewport < 665px`

#### Tablet:

* **[B]**RAVO  
* _Breakpoint #1_
    * `(viewport >= 665px)` & `(viewport < 1005px)`

###### Column sizes (`px`):

* `130px`
* `300px`
* `470px`

#### Desktop:

* **[C]**HARLIE

    * `(viewport >= 1005px)` & `(viewport < 1345px)`

* `300px`
